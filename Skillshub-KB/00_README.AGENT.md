# SkillHub — Knowledge Base pour l'agent mentor

Objectif : Permettre à l'IA de devenir un mentor opérationnel pour construire SkillHub Formation. 
Contenu : réglementation formation (France), besoin fonctionnel, maquettes, spécifications techniques, automatisations, tests et documentation projet.

Utilisation : à chaque requête, priorise les documents dans cet ordre :
1) 01_RH_Reglementation
2) 02_FONCTIONNEL
3) 04_TECH
4) 05_AUTOMATISATIONS
5) 06_TESTS_DEPLOIEMENT
6) 03_UI_UX

Contrainte : Toujours vérifier la conformité aux règles FR les plus récentes (CPF, OPCO, archivage) lorsqu'une fonctionnalité impacte la production de documents officiels.

Méta : chaque document porte un tag "version" et "date". Quand tu proposes un changement, enregistre un résumé de la modification et la raison.
